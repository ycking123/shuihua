# 🚀 快速启动指南 (Windows)

## 1. 环境准备
确保已安装以下软件：
- **Python 3.8+** (建议使用 Conda)
- **Node.js 16+**
- **MySQL 8.0+** (确保服务已启动)
- **Google Chrome** (用于爬虫)

## 2. 安装依赖
### 后端依赖
```powershell
# 激活 Conda 环境
conda activate water-essence-sprite  # 或你的环境名

# 安装 Python 包
pip install -r server/requirements.txt
pip install -r backend/requirements.txt
# 注意：如果是开发环境，还需要安装 playwright 浏览器
playwright install chromium
```

### 前端依赖
```powershell
npm install
```

## 3. 配置文件 (.env)
确保项目根目录下有 `.env` 文件，并填写了必要的 API Key 和数据库配置。

## 4. 一键启动 (推荐)
已为您创建 Windows 启动脚本，可一键启动所有服务（主后端、企微后端、前端）。

**方法 A：直接运行脚本 (PowerShell)**
```powershell
./run_project.ps1
```

**方法 B：双击运行**
直接双击根目录下的 `run_project.bat` 文件。

脚本会自动打开三个新窗口分别运行：
1. **Main Server** (Port 8000): 主业务逻辑
2. **WeChat Worker** (Port 8080): 企微消息接收与爬虫服务
3. **Frontend** (Port 5173): React 前端页面

## 5. 手动启动 (备选)
如果脚本无法运行，可在三个独立的终端中分别运行：

**终端 1 (主后端):**
```powershell
$env:PYTHONPATH=".;server"
python -m server.main
```

**终端 2 (企微后端):**
```powershell
$env:PYTHONPATH=".;server"
python -m backend.server_receive
```

**终端 3 (前端):**
```powershell
npm run dev
```
